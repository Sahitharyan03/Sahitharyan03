## 🚀 What I'm Actually Building

<table>
<tr>
<td width="50%" valign="top">

### 🌉 ContextBridge AI
> *Transfer conversations between Claude, ChatGPT & Gemini — without losing context.*

An open-source **AI context continuity engine** built as a Chrome Extension + FastAPI backend. Uses a local **Ollama LLM** (llama3.2:3b) + **spaCy semantic scoring** to extract the most important messages, compress them intelligently, and generate a bridging prompt for any target AI platform.

**MVP shipped in 3 days. Full pipeline verified live.**

`JavaScript · Python · FastAPI · Ollama · spaCy · React · Chrome MV3`
